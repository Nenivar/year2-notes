Algorithmic Strategies

Iteration

  1. Process inputs 1,...,n−11,...,n-11,...,n−1
  2. Combine output with input nnn

e.g. Insertion Sort

Typical reccurence for worst-case time:

T(n)=aT(n−1)+f(n)T(n) = aT(n-1) + f(n) T(n)=aT(n−1)+f(n)

Divide-And-Conquer

  1. Divide inputs 1,...,n−11,...,n-11,...,n−1 into a fixed number of groups and inputs
  2. Process each group of inputs
  3. Combine together the results

e.g. Merge Sort

Typical reccurence for worst-case time:

T(n)=aT(n/a)+f(n)T(n) = aT(n/a) + f(n) T(n)=aT(n/a)+f(n)
